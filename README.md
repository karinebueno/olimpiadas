# Como Usar Esta Aplicação

**Objetivo:**

Esta aplicação permite pesquisar atletas e esportes, fornecendo informações detalhadas sobre cada um.

**Funcionalidades:**

* **Pesquisa:** Digite o nome de um atleta ou esporte no campo de pesquisa.
* **Resultados:** A aplicação exibirá os resultados relevantes, incluindo o nome do atleta ou esporte, uma breve descrição e um link para mais informações.

**Como Usar:**

1. **Digite a palavra-chave:** Insira o nome do atleta ou esporte que você deseja pesquisar no campo de pesquisa.
2. **Clique em Pesquisar:** A aplicação exibirá os resultados relevantes.
3. **Clique em "Mais informações"** para acessar a página do atleta ou esporte.

**Tecnologias Utilizadas:**

* **HTML:** Estrutura básica da página.
* **CSS:** Estilização da página.
* **JavaScript:** Lógica da aplicação, incluindo a pesquisa e a exibição dos resultados.

**Contribuições:**

Sinta-se à vontade para contribuir com melhorias, correções de bugs ou novas funcionalidades.
